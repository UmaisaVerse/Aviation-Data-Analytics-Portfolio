# Project 02 — Airline Staff & Attrition Analysis

## Overview
Analysis of employee attrition data for an airline company to identify
which departments and roles have the highest staff turnover.

## Tool
Microsoft Excel

## Dataset
IBM HR Analytics Employee Attrition Dataset — Kaggle (1,470 rows, 35 columns)
Adapted to airline-specific context.

## Key Skills Demonstrated
- Data Cleaning (duplicates, blanks, inconsistencies)
- Find & Replace (airline-specific renaming)
- Nested IF Formulas
- VLOOKUP
- AVERAGEIF
- Pivot Tables
- Conditional Formatting
- Dashboard Design

## Key Findings
- Commercial & Revenue has highest department attrition at 20.63%
- Ticketing Agents are most at-risk role at 39.76%
- Overall company attrition rate: 16.12%
