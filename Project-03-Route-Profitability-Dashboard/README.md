# Project 3: Advanced Route Profitability Dashboard

Tool: Microsoft Excel
Dataset: Kaggle Airline Dataset (98,619 flights)

## Goal
Identify profitable and unprofitable airline routes after factoring in fuel, staff, and operational costs.

## Key Findings
- Most Profitable Route: Böblingen Flugfeld (DE) → PHM
- Least Profitable Route: Falcon State Airport (US) → FAL
- Fuel = 50% of all operating costs
- Long Haul generates 2.5x more profit per flight
