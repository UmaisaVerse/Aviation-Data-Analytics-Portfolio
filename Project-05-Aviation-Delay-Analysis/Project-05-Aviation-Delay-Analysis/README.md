# Project 5: Aviation Delay Analysis
**Tool:** MySQL  
**Dataset:** U.S. Flight Delays 2013–2023 (171,666 rows)  

## Business Question
Which airlines and airports have the most delays and why?

## Key Findings
- Frontier Airlines has the worst delay rate at 25.14%
- Southwest has the most total delays (2.4M) but a moderate rate of 19.65%
- Late aircraft is the #1 delay cause (4M+ incidents)
- Rafael Hernandez Airport (Puerto Rico) has the worst airport delay rate at 31.54%
- United Airlines at Rafael Hernandez is the worst airline+airport combo at 38.51%
- 2020 saw a sharp drop to 9.21% delay rate due to COVID-19 — an anomaly, not improvement

## Skills Used
- MySQL database creation and data import
- Data cleaning (NULL checks, duplicate removal)
- Aggregation with SUM, COUNT, ROUND
- Grouping and filtering with GROUP BY, HAVING, WHERE
- Trend analysis over time
